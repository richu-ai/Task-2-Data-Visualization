# Task 2 - Data Visualization and Storytelling

## Objective
Created a Power BI dashboard using Superstore dataset to analyze sales performance and business insights.

## Tools Used
- Power BI Desktop
- Superstore Dataset

## Key Insights
- West region has highest sales.
- Profit varies across categories.
- Higher discounts tend to reduce profit.
- Sales show increasing trend over years.

## Conclusion
This dashboard converts raw sales data into meaningful insights through data storytelling and visualization.
